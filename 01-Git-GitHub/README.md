---

# Conceitos

## Git

Sistema de controle de versões utilizado para registrar alterações realizadas em arquivos.

## GitHub

Plataforma que hospeda repositórios Git e facilita colaboração entre desenvolvedores e equipes.

## Repositório

Local onde ficam armazenados os arquivos, documentação e histórico de versões de um projeto.

## Commit

Registro permanente de uma alteração realizada no projeto.

## README

Documento inicial que explica o objetivo e a organização do projeto.
